# Claude Code 프로젝트 규칙

## MD 파일 작성 규칙

MD 파일을 생성하거나 업데이트할 때 아래 규칙을 항상 적용한다.

### 1. 언어
- 모든 MD 파일은 **한국어**로 작성한다.
- 기술 용어(LLM, RAG, API 등)는 영어 원문을 그대로 사용해도 된다.

### 2. 문서 상단 필수 항목
- 제목 바로 아래에 작성일을 명시한다.
  ```
  > 작성일: YYYY-MM-DD
  ```

### 3. 목차
- 섹션이 3개 이상인 경우 문서 상단(작성일 아래)에 목차를 추가한다.
  ```markdown
  ## 목차
  - [1. 섹션명](#1-섹션명)
  - [2. 섹션명](#2-섹션명)
  ```

### 4. 이미지 및 링크 (웹 조사 시)
- 웹에서 자료를 조사한 경우 이해에 도움이 되는 이미지를 찾아 본문에 삽입한다.
- 이미지는 반드시 출처를 함께 표기한다.
  ```markdown
  ![이미지 설명](이미지_URL)
  > *출처: 사이트명 — [글 제목](URL)*
  ```
- 본문 내용과 관련된 참고 링크는 해당 섹션 안에 인라인으로 추가한다.

### 5. 참고자료 섹션
- 문서 마지막에 `## 참고자료` 섹션을 추가한다.
- 조사에 사용한 모든 출처와 이해에 도움이 되는 링크를 목록으로 정리한다.
  ```markdown
  ## 참고자료
  - [글 제목](URL) — 한 줄 설명
  ```
