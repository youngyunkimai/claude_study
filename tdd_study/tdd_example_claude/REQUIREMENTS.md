# Calculator C Library — Requirements

## 개요

정수 사칙연산을 제공하는 C 라이브러리.
에러는 반환값으로 표현한다 (Linux 커널 스타일: `0` = 성공, 음수 = 에러).

---

## 에러 코드

| 코드 | 값  | 설명             |
|------|-----|------------------|
| `0`  | 0   | 성공             |
| `-EINVAL` | -22 | 잘못된 인자 (NULL 포인터 등) |
| `-EDOM`   | -33 | 연산 도메인 오류 (0으로 나누기) |

> `errno.h` 표준 상수 사용. 결과는 포인터 매개변수(`int *result`)로 반환한다.

---

## API

```c
int calc_add(int a, int b, int *result);
int calc_sub(int a, int b, int *result);
int calc_mul(int a, int b, int *result);
int calc_div(int a, int b, int *result);
```

- 반환값: `0` (성공) 또는 음수 에러 코드
- `result`: 연산 결과를 저장할 포인터. `NULL`이면 `-EINVAL` 반환

---

## 요구사항 상세

### 1. 덧셈 — `calc_add(a, b, result)`

| # | 구분       | 입력 (`a`, `b`)     | 기대 반환값 | 기대 `*result` | 설명                   |
|---|------------|---------------------|-------------|----------------|------------------------|
| 1 | 정상       | `1`, `2`            | `0`         | `3`            | 기본 덧셈              |
| 2 | 정상       | `0`, `0`            | `0`         | `0`            | 영점 덧셈              |
| 3 | 정상       | `-3`, `5`           | `0`         | `2`            | 음수 + 양수            |
| 4 | 정상       | `-3`, `-4`          | `0`         | `-7`           | 음수 + 음수            |
| 5 | 경계       | `INT_MAX`, `0`      | `0`         | `INT_MAX`      | 최댓값 + 0             |
| 6 | 경계       | `INT_MIN`, `0`      | `0`         | `INT_MIN`      | 최솟값 + 0             |
| 7 | 에러       | (any), `NULL` result | `-EINVAL`  | —              | NULL 포인터            |

### 2. 뺄셈 — `calc_sub(a, b, result)`

| # | 구분       | 입력 (`a`, `b`)     | 기대 반환값 | 기대 `*result` | 설명                   |
|---|------------|---------------------|-------------|----------------|------------------------|
| 1 | 정상       | `5`, `3`            | `0`         | `2`            | 기본 뺄셈              |
| 2 | 정상       | `0`, `0`            | `0`         | `0`            | 영점 뺄셈              |
| 3 | 정상       | `3`, `5`            | `0`         | `-2`           | 결과가 음수            |
| 4 | 정상       | `-3`, `-4`          | `0`         | `1`            | 음수 - 음수            |
| 5 | 경계       | `INT_MIN`, `0`      | `0`         | `INT_MIN`      | 최솟값 - 0             |
| 6 | 경계       | `0`, `INT_MIN`      | `0`         | `INT_MIN + 1`  | ※ 실제 오버플로 없음   |
| 7 | 에러       | (any), `NULL` result | `-EINVAL`  | —              | NULL 포인터            |

### 3. 곱셈 — `calc_mul(a, b, result)`

| # | 구분       | 입력 (`a`, `b`)     | 기대 반환값 | 기대 `*result` | 설명                   |
|---|------------|---------------------|-------------|----------------|------------------------|
| 1 | 정상       | `3`, `4`            | `0`         | `12`           | 기본 곱셈              |
| 2 | 정상       | `0`, `999`          | `0`         | `0`            | 0 곱하기               |
| 3 | 정상       | `-3`, `4`           | `0`         | `-12`          | 음수 × 양수            |
| 4 | 정상       | `-3`, `-4`          | `0`         | `12`           | 음수 × 음수            |
| 5 | 경계       | `1`, `INT_MAX`      | `0`         | `INT_MAX`      | 1 × 최댓값             |
| 6 | 경계       | `0`, `INT_MIN`      | `0`         | `0`            | 0 × 최솟값             |
| 7 | 에러       | (any), `NULL` result | `-EINVAL`  | —              | NULL 포인터            |

### 4. 나눗셈 — `calc_div(a, b, result)`

| # | 구분       | 입력 (`a`, `b`)     | 기대 반환값 | 기대 `*result` | 설명                        |
|---|------------|---------------------|-------------|----------------|-----------------------------|
| 1 | 정상       | `10`, `2`           | `0`         | `5`            | 기본 나눗셈                 |
| 2 | 정상       | `7`, `2`            | `0`         | `3`            | 소수점 버림 (정수 나눗셈)   |
| 3 | 정상       | `0`, `5`            | `0`         | `0`            | 0 ÷ 양수                    |
| 4 | 정상       | `-9`, `3`           | `0`         | `-3`           | 음수 ÷ 양수                 |
| 5 | 경계       | `INT_MIN`, `1`      | `0`         | `INT_MIN`      | 최솟값 ÷ 1                  |
| 6 | 경계       | `INT_MAX`, `INT_MAX`| `0`         | `1`            | 같은 값으로 나누기          |
| 7 | 에러       | `10`, `0`           | `-EDOM`     | —              | 0으로 나누기                |
| 8 | 에러       | (any), `NULL` result | `-EINVAL`  | —              | NULL 포인터                 |

## 비기능 요구사항

| 항목         | 내용                                              |
|--------------|---------------------------------------------------|
| 언어 표준    | C99 이상                                          |
| 정수 타입    | `int` (플랫폼 네이티브 32-bit 가정)               |
| 오버플로     | 미정의 동작 허용 (이번 범위에서 오버플로 테스트 제외) |
| 헤더         | `calculator.h` 에 API 선언                        |
| 테스트 프레임워크 | Unity (또는 동등한 C 단위 테스트 프레임워크)  |
| 빌드 시스템  | Makefile                                          |
